HELP: Quick System Info (gui)

Displays a report based on inxi and a few custom parameters for MX Linux.

Use the "Copy for Forum" button at the bottom of the report display to copy the report to the clipboard in a format suitable

for directly pasting into a forum post, complete with code tags

Use the "Save" button to save the report to a text file

Shortcut keys: Alt+C plain text copy, CTRL+C Copy for forum

Development history: MX Linux
